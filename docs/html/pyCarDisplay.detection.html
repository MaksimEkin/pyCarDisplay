pyCarDisplay.detection.object_detection_api module¶

This is the object detection module.

This code is borrowed from sgrvinod's GitHub repository named a-PyTorch-Tutorial-to-Object-Detection, and modified to work with pyCarDisplay. It preserved the main functionality of the unmodified version.

Reference:

Vinodababu, S. (n.d.). A-PyTorch-Tutorial-to-Object-Detection. https://github.com/sgrvinod/a-PyTorch-Tutorial-to-Object-Detection
